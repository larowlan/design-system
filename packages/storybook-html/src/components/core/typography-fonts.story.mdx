## Loading fonts

Copy the HTML or CSS below and paste it into your project to load design system fonts.

**HTML**
```html
<link rel="stylesheet" type="text/css" href="https://static.uq.net.au/v13/fonts/Roboto/roboto.css" />
<link rel="stylesheet" type="text/css" href="https://static.uq.net.au/v13/fonts/Merriweather/merriweather.css" />
<link rel="stylesheet" type="text/css" href="https://static.uq.net.au/v13/fonts/Montserrat/montserrat.css" />
```

**CSS**
```css
@import url('https://static.uq.net.au/v13/fonts/Roboto/roboto.css');
@import url('https://static.uq.net.au/v13/fonts/Merriweather/merriweather.css');
@import url('https://static.uq.net.au/v13/fonts/Montserrat/montserrat.css');
```

## Font stacks

The following font family declarations use widely available fonts. A user's web browser will look at the comma-separated list and will
attempt to use each font in turn until it finds one that is available on the user's computer. The final "generic" font (sans-serif or serif) hints at what type of font to use if the web browser doesn't
find any of the fonts in the list.

**Roboto**
```css
font-family: "Roboto", "Helvetica Neue", Helvetica, Arial, sans-serif;
```
**Merriweather**
```css
font-family: Merriweather, Georgia, serif;
```
**Montserrat**
```css
font-family: Montserrat, "Helvetica Neue", Helvetica, Arial, sans-serif;
```

## Alternative fonts

When online platforms and systems are not able to display our fonts such as third party applications, use our
alternative fonts. If none of the alternative font options are available, it is recommended to use the default font provided by the platform or system.
