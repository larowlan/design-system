# Typography
At UQ, we have a set of complementary font assets and typeface rules that ensure a consistent look and feel of online content across devices and computers.

## UQ's Fonts

Web based fonts are used for websites and applications, where the font can be embedded into code to display seamlessly. 
The 3 fonts used for UQ online applications are Roboto, Merriweather & Montserrat.

### Font stacks

The following font family declarations use widely available fonts.
A user's web browser will look at the comma-separated list and will
attempt to use each font in turn until it finds one that is available
on the user's computer. The final "generic" font (sans-serif, serif or
monospace) hints at what type of font to use if the web browser doesn't
find any of the fonts in the list.

Roboto
```css
font-family: "Roboto", "Helvetica Neue", Helvetica, Arial, sans-serif;
```
Merriweather
```css
font-family: Merriweather, Georgia, serif;
```
Montserrat
```css
font-family: Montserrat, "Helvetica Neue", Helvetica, Arial, sans-serif;
```

Monospace (Based on available system fonts across common OS's and devices)
```css
font-family: Consolas, "Andale Mono WT", "Andale Mono", "Lucida Console", "Lucida Sans Typewriter", "DejaVu Sans Mono", "Bitstream Vera Sans Mono", "Liberation Mono", "Nimbus Mono L", Monaco, "Courier New", Courier, monospace;
```

### Loading fonts, HTML, CSS

Copy the HTML or CSS lines below and paste it into your project to load design system fonts.

#### HTML
```html
<link rel="stylesheet" type="text/css" href="https://static.uq.net.au/v13/fonts/Roboto/roboto.css" />
<link rel="stylesheet" type="text/css" href="https://static.uq.net.au/v13/fonts/Merriweather/merriweather.css" />
<link rel="stylesheet" type="text/css" href="https://static.uq.net.au/v13/fonts/Montserrat/montserrat.css" />
```

#### CSS
```css
@import url('https://static.uq.net.au/v13/fonts/Roboto/roboto.css');
@import url('https://static.uq.net.au/v13/fonts/Merriweather/merriweather.css');
@import url('https://static.uq.net.au/v13/fonts/Montserrat/montserrat.css');
```

## Alternative fonts

When online platforms and systems are not able to display our brand font (ie. When embedding fonts is not possible) such as a third party application. In these instances, use our
alternative fonts if available. If none of the preferred options are available it is recommended to use the default font provided, ***Do not choose random alternative fonts***. 

### Arial

It can be used for most heading and body copy needs. This font is available as a standard PC and Mac font.

### Georgia

Georgia can be used for pull out quotes and to break up long headings. ***It is not to be used for body text***.
