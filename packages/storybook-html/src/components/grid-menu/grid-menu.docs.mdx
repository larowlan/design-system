# Grid menu

<StatusBadge.Review /><StatusBadge.DraftDoc />

## Overview

Grid menu is a list of unordered list elements that can be displayed in either 1, 2, 3 or 4 columns. 
By default it is displayed as a 3 column layout and changes on different responsive breakpoints.

There are broadly 2 categories of Grid menus. 
One where the height of the list elements is by default set to 'auto' and the other where all the list element heights are equalised via JS.

### b) Grid menu - height equalised
In this category of the grid menu component, if the height of an item is bigger the alignment of the items are taken care of via JS.

<Story id="components-grid-menu--equalised" />

