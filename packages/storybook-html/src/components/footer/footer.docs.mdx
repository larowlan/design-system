# Global footer

<StatusBadge.Review /><StatusBadge.DraftDoc />

## Overview

The global footer provides a consistent branding and navigational element across all UQ pages. It contains supplementary information such as copyright, ABN details, contact and emergency information, links to other pages within the website, and the Reconciliation statement.
