# Global header

<StatusBadge.Review /><StatusBadge.DraftDoc />

The global header provides a consistent branding and navigational element across
all UQ pages. Reinforced by the UQ strapline, the header enables users to
quickly identify and trust UQ websites.

## With global search only

Provides the UQ strapline, global links, and a general UQ site search feature
using [*UQ Search*](https://www.uq.edu.au/search).

> Don't reappropriate the search function for special integrated search
> functionality within your application e.g. searching for a location on a map,
> searching for books or articles in a library database.

## With global and site search

Provides the UQ strapline, global links, and a site search feature using
[*UQ Search*](https://www.uq.edu.au/search). Two radio options are included to
allow users to choose to search all UQ sites or just the content on your site\*.

> \*If your application/website isn't indexed by UQ Search, please opt
> to use the global header **with global search only**.
