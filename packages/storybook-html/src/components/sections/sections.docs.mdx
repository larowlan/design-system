## Overview

Sections are a layout style that wrap content in a white or grey background with a range of content-width, content-alignment, and padding options. 

<><br /></>

## Default

The default section includes left-aligned content on a white background that is contained to the page width. 

<Canvas withSource="none">
  <Story id="layout-sections--sections-default" />
</Canvas>

```html
<div class="uq-section">
  <div class="uq-section__container">
    <div class="uq-section__content">
      <!--section content-->
    </div>
  </div>
</div>
```

<><br /></>

### Spacing

**Top and bottom spacing** - Sets the amount of padding at the top and bottom
- **No spacing** `uq-section--no-spacing` - removes vertical padding.
- **Large spacing** `uq-section--large-spacing` - adds large spacing top and bottom.
- **Double spacing** `uq-section--double-spacing` - alias of `uq-section--large-spacing`
- **Extra Large spacing** `uq-section--extra-large-spacing` - adds extra large spacing top and bottom.

**Top spacing** - Sets the amount of padding at the top of the section.
- **No spacing top** `uq-section--no-top-spacing` - removes top padding only.
- **large spacing top** `uq-section--large-top-spacing` - adds large spacing above to show more background image.
- **Extra large spacing top** `uq-section--extra-large-top-spacing` - adds extra large spacing above.

**Bottom spacing** - Sets the amount of padding at the bottom of the section.
- **No spacing bottom** `uq-section--no-bottom-spacing` - removes bottom padding only.
- **large spacing bottom** `uq-section--large-bottom-spacing` - adds large spacing below to show more background image.
- **Extra large spacing bottom** `uq-section--extra-large-bottom-spacing` - adds extra large spacing below.

## How and when to use

Sections are primarily used on landing pages to create strips of content stacked vertically down the page, while containing the content to the same page column width.
