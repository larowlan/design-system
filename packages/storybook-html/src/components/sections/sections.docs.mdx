# Sections

## Overview

Sections are a layout style that wraps content. They provide a header with title and summary, and a footer field along with a range of themeing options. They are primarily used on landing pages to create strips of content stacked vertically down the page, while containing the content to the same page column width.

## Default Section

```html
<div class="section">
  <div class="section__container">
    <div class="section__content">
      <!--section content-->
    </div>
  </div>
</div>
```

## Optional content blocks

- **Title** - Optional. `.section__title`
- **Summary** - Optional. `.section__summary` Short summary text to accompany the section title and give context to the content below.
- **Footer** - Optional. `.section__footer` Short text that displays below the section content.

```html
<div class="section">
  <div class="section__container">
    <div class="section__header">
      <h2 class="section__title">Section header title</h2>
      <p class="section__summary">Section header summary.</p>
    </div>
    <div class="section__content">
      <p>section content.</p>
    </div>
    <div class="section__footer">section footer.</div>
  </div>
</div>
```
## Section layouts

**Single column, left-aligned** - Section header and footer are stacked above and below the section content, and are left aligned. The section content alignment is not affected by this setting.

### Centered

**Single column, centered** - Section header and footer are stacked above and below the section content, and are centered. The section content alignment is not affected by this setting.

Centered layout `section--centered`

### Horizontal

**Two column, left-aligned** - Section header is shifted to the left of the section content and displayed in a 4/8 column arrangement.

## Alternative Styles
These settings allow a range of styling to be applied to the section. These include

**Default** - Sections have a white background by default.

### Shaded

**Shaded background** - Applies a light grey background to the section.

Shaded Style `section--shaded`

  <Story id="layout-sections--sections-shaded" />

### Background image

**Background image** - Optional. Applies the chosen image as a background to the section. It will be fitted to fully cover the section so it is recommended that the uploaded image be at least 1920px wide, and as tall as is needed to cover the amount of content in the section.

Background image `section--image-behind`
Scale background images on the section to always cover `section--background-image`

<Canvas withSource="none">
  <Story id="layout-sections--sections-background" />
</Canvas>

```html
<div class="section section--image-behind">
  <div class="section__background"><img src="http://lorempixel.com/1920/1080" /></div>
  </div>
</div>
```

### Light tint and dark text

**Light tint** - Applies a light tint overlay to the section. As with the dark tint, this works in the opposite way, lightening the background image for better contrast with dark text on top.

Light tint `section--light-tint` 

**Dark text** - Sets the section title, summary, and footer text to dark grey. For use in conjunction with the light tint option.

Dark text `section--dark-text`

### Dark tint and light text

**Dark tint** - Applies a dark tint overlay to the section. This is for use with a background image so that the background image can be softly muted to allow content on top to display more clearly.

Dark tint `section--dark-tint` 

**Light text** - Sets the section title, summary, and footer text to white. For use in conjunction with the dark tint option.

### Spacing

**Top and bottom spacing** - Sets the amount of padding at the top and bottom
- **No spacing** `section--no-spacing` - removes vertical padding.
- **Large spacing** `section--large-spacing` - adds large spacing top and bottom.
- **Double spacing** `section--double-spacing` - alias of `section--large-spacing`
- **Extra Large spacing** `section--extra-large-spacing` - adds extra large spacing top and bottom.

**Top spacing** - Sets the amount of padding at the top of the section.
- **No spacing top** `section--no-top-spacing` - removes top padding only.
- **large spacing top** `section--large-top-spacing` - adds large spacing above to show more background image.
- **Extra large spacing top** `section--extra-large-top-spacing` - adds extra large spacing above.

**Bottom spacing** - Sets the amount of padding at the bottom of the section.
- **No spacing bottom** `section--no-bottom-spacing` - removes bottom padding only.
- **large spacing bottom** `section--large-bottom-spacing` - adds large spacing below to show more background image.
- **Extra large spacing bottom** `section--extra-large-bottom-spacing` - adds extra large spacing below.
